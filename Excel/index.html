Text to column

Text to column is really useful if you need to split information in a cell up, eg if you are working with geographical info etc or if you need to split out words for whatever reason. Usually you split by space but you can split by any single character you want.

  • Right click on column D and press insert, this should add a blank column
  • Highlight column C ("Owner name")
  • Select Data from the top menu, then Text to columns

VLOOKUP Function

If you type in =VLOOKUP in Excel it will prompt you with this:

=VLOOKUP(lookup_value, table_array, col_index_num, [range_lookup])

What this means is: Look for this specific value, in this selection of data, match it with this column, and either look for a true match or a fuzzy match.

Some notes: VLOOKUP can be a bit fiddly sometimes. Make sure that your lookup_value (the value you want to check against) is in the first column of the table_array (the data range you're looking in).
